# 快餐连锁大亨 精简规则书

本规则书整合了游戏的基础规则、常见问题解答（FAQ）以及“番茄酱机制与更多想法”扩展包中的所有模块。阅读本规则后，您无需查阅原始PDF文件即可大致了解完整的游戏设置与玩法。

## 1. 游戏概述

《快餐连锁大亨》是一款关于建立快餐连锁店的重度策略游戏 。玩家从一个企业家开始，通过招聘和管理员工（由卡牌代表）来建立自己的公司 。玩家们在一个可变的城市地图上，通过采购、营销和销售来进行竞争 。游戏会持续进行，直到银行第二次宣告破产 。游戏结束时，最富有的玩家获胜 。

## 2. 核心概念

### 员工卡 (Employee Cards)

- 功能: 卡牌下方文字描述了该员工在公司中的职能 。

- 招聘: 左上角有 “1” 标志的卡牌是入门级员工，可以通过招聘获得 。

- 薪水: 卡牌上有 "$" 标志的员工需要支付薪水 。

- 培训: 左下角显示了该员工可以被培训成的更高级职位 。

- 唯一职位 (1x): 左下角有 "1x" 标志的卡牌是高级职位，每位玩家对每种该类卡牌只能拥有一张 。

- 经理: 黑色背景的卡牌是经理 。

- 距离: 右上角可能标有该员工行动的“距离”范围，分为公路或飞艇两种计算方式 。

### 公司结构 (Company Structure)

- 在“重组公司”阶段，玩家需要将“在岗”的员工卡牌以CEO为顶点，排列成金字塔结构 。

- CEO位于顶端，直接向他汇报的卡牌不能超过其“卡槽”数量（初始为3个） 。

- 向CEO汇报的可以是普通员工或经理 。

- 经理只能向CEO汇报 。向经理汇报的只能是普通员工，不能是其他经理 。经理拥有的卡槽数量（2-10个）标注在卡牌上 。

- 忙碌的营销员不被放置在公司结构中，也不占用任何卡槽 。

### 地图与距离 (Map & Range)

- 地图板块: 游戏地图由多个5x5格的正方形板块拼接而成 。

- 距离计算: 距离（Range）指的是跨越地图板块边界的次数 。距离通常沿着公路计算，除非卡牌上有飞艇标志 。

- 从餐厅计算距离时，总是从其入口处开始计算 。

### 餐厅 (Restaurants)

- 每个餐厅占据2x2的方格 。其中一个角落有门的方格被视为其“入口” 。

### 食物与饮品 (Food & Drinks)

- 木制配件代表食物（汉堡、披萨）和饮品（软饮、柠檬水、啤酒） 。

- 放置在房屋上时，它们代表“需求” 。放置在营销广告牌上时，代表广告内容和持续时间 。在玩家面前时，代表玩家的“库存” 。

- 食物和饮品配件是无限供应的 。

- 大号的食物和饮品配件代表5个对应的小配件 。

### 金钱 (Money)

- 游戏中的金钱是公开信息，无需隐藏 。

- 纸币是无限供应的，但银行中的钱是有限的，会影响游戏进程 。

### 组件限制 (Component Limitations)

- 有限组件: 员工卡、房屋、花园、餐厅和营销板块等所有纸板组件，数量均受游戏提供的配件限制 。

- 无限组件: 食物和饮品配件、里程碑卡是无限供应的 。如果里程碑卡用完，可以用替代物来记录 。

## 3. 游戏设置

### 卡牌设置 (Card Setup)

- 将所有员工卡和里程碑卡分类面朝上放在桌面中央，以便所有玩家都能看到 。

- 根据玩家人数，调整“1x”员工卡的数量：

  - 2-3人游戏: 每种“1x”卡使用1张 。

  - 4人游戏: 每种“1x”卡使用2张 。

  - 5-6人游戏: 每种“1x”卡使用3张 。

### 地图设置 (Map Setup)

- 根据玩家人数，随机抽取地图板块，并以随机朝向拼成一个矩形网格 。

  - 2人游戏: 3x3 。

  - 3人游戏: 3x4 。

  - 4人游戏: 4x4 。

  - 5人游戏: 5x4 。

  - 6人游戏: 4x6 (必须使用“新区域”扩展模块) 。

- 根据下表，将指定编号的广告牌营销板块移出游戏：

  - 2人游戏: 移除 #12, #15, #16 。

  - 3人游戏: 移除 #15, #16 。

  - 4人游戏: 移除 #16 。

  - 5人游戏: 不移除 。

### 银行设置 (Bank Setup)

- 将 玩家人数 x $50 的现金放入银行区作为初始资金 。

- 玩家在游戏开始时不获得任何金钱 。

### 玩家设置 (Player Setup)

- 每位玩家选择一个连锁餐厅品牌，并领取对应的1张CEO卡、3个餐厅模型和一套3张银行储备卡 。

### 初始顺序设置 (Initial Turn Order)

- 收集所有玩家的顺序标记，随机抽取并依次放置在顺序轨上，决定初始游戏顺序 。

### 放置初始餐厅 (Placing Initial Restaurants)

- 从顺序轨上最后一位的玩家开始，逆序放置初始餐厅，第一位的玩家最后放置 。

- 初始放置餐厅有以下限制：

  - 必须完全放置在空方格上 。

  - 餐厅的入口必须与一个有公路的方格相邻 。

  - （仅限初始设置） 餐厅入口所在的地图板块上，不能有其他餐厅的入口 。只要入口在不同板块上，两个餐厅可以部分重叠在同一个板块上 。

### 设置储备卡 (Setting Reserve Cards)

- 每位玩家秘密选择一张自己的银行储备卡，面朝下地放在银行旁边 。这些卡牌决定了当银行第一次破产时会补充多少资金以及CEO卡槽可能发生的变化 。

## 4. 游戏流程（各阶段详解）

游戏按回合进行，每回合包含七个阶段。

### 阶段一：重组公司 (Phase 1: Restructuring)

1. 所有玩家同时秘密地将自己手中的员工卡分为两堆：本回合“在岗”和“待命” 。
2. CEO卡总是“在岗” 。
3. 所有玩家同时展示“在岗”的员工卡，并将它们按照公司结构规则排列在CEO下方 。
4. 如果放置的员工数量超出了公司结构可容纳的上限，除了CEO外的所有员工都将变为“待命”状态 。

### 阶段二：决定顺序 (Phase 2: Order of Business)

1. 根据公司结构中的“空余卡槽”数量决定选择顺序，拥有最多空余卡槽的玩家最先选择 。
2. 拥有“首个飞机营销”里程碑的玩家，在计算时额外增加2个空余卡槽 。
3. 如果空余卡槽数量相同，则上一回合顺序靠前的玩家先选 。
4. 玩家从顺序轨上选择一个空位，放置自己的顺序标记，这决定了本回合的行动顺序 。

### 阶段三：工作时间 (Phase 3: Working 9-5)

玩家按照新的顺序轮流执行行动 。每位玩家在自己的回合内，完成自己所有“在岗”员工的行动 。行动必须按以下子阶段顺序执行 。

- 强制与可选行动: 定价经理、折扣经理、奢侈品经理、CFO、招聘经理、人力资源总监和女服务员的行动是强制的；其他所有员工的行动是可选的 。

#### 子阶段 1: 招聘 (Recruit)

- 招聘次数：等于你所有在岗“可招聘员工”（员工数据 `usage_tags` 包含 `use:recruit`）的 `recruit_capacity` 之和（默认 CEO=1）。

- 每次招聘行动可以从供应区拿取一张入门级员工卡（左上角有“1”标志），并将其放入你的“待命”区 。

- 即使某个员工牌堆已空，你仍然可以招聘该员工，但前提是在紧接着的培训阶段立即将其培训成更高级的职位 。这个操作本身并不会提前开始你的培训阶段 。

#### 子阶段 2: 培训 (Train)

- 每次培训行动可以培训一名“待命”区的员工 。

- 将被培训的卡牌放回供应区，并拿取其目标职位的新卡牌 。

- 招聘和培训是两个独立的行动 。

- 培训时，只需要最终职位有卡牌可用即可，中间过程的职位卡牌可以缺货 。

#### 子阶段 3: 发起营销活动 (Initiate Marketing Campaigns)

- 每位在岗的营销员可以发起一次营销活动 。

- 类型、距离、时长: 取决于营销员卡牌的描述 。距离从你任何一个餐厅的入口开始计算 。

- 放置:

  - 广告牌、邮箱和收音机必须放置在地图上的空格上，且必须与一条公路相邻 。对于收音机，它可以放置在地图上任何一条公路旁边的空格，这条公路无需与你的任何餐厅相连 。

  - 飞机则放置在游戏区域的边缘 。

- 广告内容: 选择一种食物或饮品作为广告内容，并将对应数量的配件（等于持续回合数）放在营销板块上 。所有营销都只针对单一产品 。

- 忙碌状态: 发起营销后，该营销员卡牌被放在一旁并用“忙碌”标记盖住，直到营销活动结束 。

#### 子阶段 4: 获取食物和饮品 (Get Food & Drinks)

- 在岗的厨房员工可以生产食物；采购员可以获取饮品 。

- 所有获得的食物和饮品都放在玩家面前，可供其所有餐厅共同使用 。

- 采购员规则:

  - 他们从餐厅出发，沿着路线移动。无需返回餐厅 。

  - 不允许U型转弯 。

  - 每个饮品标志在每个回合只能被同一个采购员拾取一次 。但如果你有多个采购员，他们可以先后从同一个标志处拾取饮品 。

#### 子阶段 5: 放置新房屋和花园 (Place New Houses and Gardens)

- 在岗的“可放置房屋/花园员工”（员工数据 `usage_tags`：`use:place_house` / `use:add_garden`）可以放置一个新的房屋板块或为一个已有的房屋增加一个花园 。

#### 子阶段 6: 放置或移动餐厅 (Place or Move Restaurants)

- 在岗的本地经理或区域经理可以放置新餐厅或移动已有餐厅 。

- 在此阶段放置餐厅时，不再受“一个地图板块只能有一个餐厅入口”的初始设置限制 。

- 如果你使用了本地或区域经理，你的所有餐厅在本回合都会自动获得“免下车”能力（即每个角落都视为入口） 。

- 除了飞机外，所有组件在放置时都可以旋转 。

### 阶段四：晚餐时间 (Phase 4: Dinnertime)

1. 处理房屋: 按照房屋上的数字从小到大的顺序，依次为每个有“需求”标记的房屋决定去向 。

2. 决定餐厅:

    - 首先，找出所有能通过公路连接到该房屋，并且库存能满足其全部需求的连锁餐厅 。

    - 若有多家满足条件，居民会选择“单价 + 距离”总和最低的餐厅 。

    - 若总和仍然相同，则选择在岗“女服务员”数量最多的餐厅 。

    - 若还是相同，则由当前回合顺序靠前的玩家获胜 。

3. 计算收入:

    - 获胜的餐厅从库存中移除相应食物/饮品，并获得收入 。

    - 收入公式为：(单价 x 售卖数量) + 额外奖励（但最终收入有下限） 。

    - 如果房屋有花园，单价部分翻倍，但额外奖励不翻倍 。

    - 售价最低为0：即使各种修正后计算结果小于0，最终收入也按0计算，不会出现负收入，也不会向银行支付费用。

4. 女服务员收入: 所有房屋处理完毕后，每位在岗的女服务员会为你赚取3（或5，如果获得里程碑） 。

5. CFO加成: 之后，拥有在岗CFO（或“拥有$100”里程碑）的玩家，其本回合获得的所有现金（包括女服务员的小费）增加50%（向上取整）。

6. 银行破产 (Breaking the Bank):

    - 第一次: 如果银行在支付期间钱不够，则第一次破产。翻开所有玩家之前选择的储备卡，将卡上标示的金额加入银行，并根据卡上出现最多次的数字决定所有CEO之后的新卡槽数 。

    - 第二次: 如果银行再次钱不够，游戏在当前晚餐阶段结束后立刻结束 。完成所有应付的款项，本回合将不再有发薪日阶段 。

### 阶段五：发薪日 (Phase 5: Payday)

1. 解雇: 你可以选择解雇任意数量的员工（包括在岗和待命的）。被解雇的卡牌放回供应区。通常忙碌的营销员不能解雇。

2. 支付薪水: 为你所有（未被解雇的）带""标志的员工，每人支付$5薪水。薪水返还给银行。所有可用的折扣必须使用。

3. 忙碌的营销员: 仍需支付薪水。特殊例外：若你已解雇了所有其他需要薪水的员工，仍没有足够金钱支付忙碌的营销员，则允许解雇该忙碌的营销员；对应的营销活动仍留在版图上。

4. 折扣: 招聘经理和人力资源总监未使用的招聘次数，可以用来抵扣薪水（每次$5）。这些折扣是强制使用的。

    - 仅在岗的招聘经理/人力资源总监计入折扣次数（待命不计入）。

5. 最低支付额: 无论折扣多少，最低支付为$0。发薪不会导致负支付或破产。

6. 支付顺序: 所有玩家同时支付薪水。

### 阶段六：营销活动 (Phase 6: Marketing Campaigns)

1. 按照营销板块上的数字从小到大的顺序，依次结算每个营销活动的效果 。

2. 需求上限: 普通房屋最多有3个需求标记，有花园的房屋最多5个 。

3. 营销范围:

    - 广告牌 (Billboard): 影响所有与之横向或纵向相邻的房屋 。

    - 邮箱 (Mailbox): 影响所有在同一个“街区”（即不跨越公路所能到达的区域）的房屋 。

    - 飞机 (Airplane): 影响其飞行路径下（一行或一列）的所有房屋 。

    - 收音机 (Radio): 影响其所在板块及周围8个板块上的所有房屋 。

4. 结算后: 从每个营销板块上移除一个代表持续时间的配件。如果没有配件了，将该营销板块和对应的“忙碌”标记收回，解放该营销员 。

### 阶段七：清理 (Phase 7: Cleanup)

1. 丢弃库存: 没有“首个丢弃食物/饮品”里程碑（即没有冰箱）的玩家，必须丢弃所有剩余的食物和饮品库存 。

2. 收回卡牌: 将所有在岗和待命的员工卡收回手中 。

3. 新店开业: 将所有“即将开业”的餐厅翻面至“欢迎光临” 。

4. 移除里程碑: 将本回合中至少被一名玩家获得的里程碑类型，从供应区中全部移除（翻面），其他玩家本局游戏无法再获得这些里程碑 。

## 5. 里程碑详解（基础游戏）

里程碑一旦达成，其效果立即生效且持续整场游戏，通常是强制性的 。在同一回合内，多名玩家可以达成并获得同一个里程碑 。

- 首个一回合雇佣三人 (First to Hire 3 People in 1 Turn): 立即获得2名管理培训生。

- 首个丢弃食物/饮品 (First to Throw Away Drink/Food): 获得一个“冰箱”，在清理阶段可以保留最多10个食物或饮品到下一回合 。

- 首个使用女服务员 (First Waitress Played): 你的每位在岗女服务员在晚餐阶段为你赚取5$，而不是3$ 。

- 首个拥有$20 (First to Have $20): 你可以立即查看面朝下的银行储备卡的内容 。

- 首个拥有$100 (First to Have $100): 你的CEO获得CFO的能力（回合收入+50%），效果从下一回合开始 。你不能再拥有或培训CFO 。

- 首个降价 (First to Lower Prices): 你的所有产品的基础单价永久 - $1 。

- 首个培训员工 (First to Train Someone): 在发薪日阶段，你的总薪水永久减少$15 。

- 首个生产汉堡 (First Burger Produced): 立即获得一名汉堡厨师 。

- 首个生产披萨 (First Pizza Produced): 立即获得一名披萨厨师 。

- 首个使用跑腿伙计 (First Errand Boy Played): 每次从饮料进货点获取的饮品数量+1 。

- 首个使用手推车操作员 (First Cart Operator Played): 你的手推车操作员、货车司机和飞艇驾驶员的距离 +1 。

- 首个支付$20+薪水 (First to Pay 20+inSalaries):你可以使用多个培训人员的培训次数来培训同一个人。

- 首个放置广告牌 (First Billboard Placed): 你的营销员不再需要支付薪水；你之后放置的所有营销活动都将变为永久性的 。

- 首个营销汉堡/披萨/饮品 (First Burger/Pizza/Drink Marketed): 每当你售出一个对应的产品，额外获得$5奖金 。该奖金不影响单价 。

- 首个营销汉堡 (First Burger Marketed): 每当你售出一个汉堡，额外获得$5奖金 。该奖金不影响单价 。

- 首个营销披萨 (First Pizza Marketed): 每当你售出一个披萨，额外获得$5奖金 。该奖金不影响单价 。

- 首个营销饮料 (First Drink Marketed): 每当你售出一个饮料，额外获得$5奖金 。该奖金不影响单价 。

- 首个飞机广告 (First Airplane Campaign): 在决定顺序阶段，计算空余卡槽时额外+2 。

- 首个电波广告 (First Radio Campaign): 你的电波广告每次会在影响的房屋上放置2个需求标记，而不是1个 。

## 7. 入门游戏建议

对于初次游戏的玩家，强烈建议采用以下简化规则，以熟悉核心机制 。
